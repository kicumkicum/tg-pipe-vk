# VK setup (Callback API)

## 1) Подготовка сообщества

- Создай/выбери VK Community (группа), в которой будет работать бот/интеграция.

## 2) Включи Callback API

- Открой настройки сообщества → **Работа с API** → **Callback API**.
- Укажи URL:
  - `https://<your-domain>/api/vk`
- Сохрани.

## 3) Secret / проверка запросов

- В настройках Callback API задай `secret` (любой длинный случайный).
- Сохрани значение в переменную окружения `VK_SECRET` в Vercel.

## 4) Confirmation (подтверждение сервера)

- VK попросит подтвердить сервер: пришлёт событие `confirmation`.
- Бэкенд должен ответить **ровно строкой** из настроек VK (без JSON, без кавычек), например: `33f381a0`.
- Сохрани эту строку в переменную окружения `VK_CONFIRMATION` в Vercel (и в `.env.local` для локали).

## 5) События

- Включи события:
  - `message_new` (достаточно для MVP “только текст”)
