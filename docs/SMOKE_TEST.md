# Smoke test

## Подготовка

- Убедись, что Vercel env vars заполнены (см. `docs/DEPLOY.md`).
- Убедись, что:
  - VK Callback API настроен на `/api/vk` и подтверждён
  - Telegram webhook настроен на `/api/telegram`

## Проверки

1) **VK → TG**
- Отправь обычный текст в VK беседе
- Убедись, что он пришёл в Telegram с префиксом `[VK | ...]` и маркером `[BRIDGE:VK→TG|...]`

2) **TG → VK**
- Отправь обычный текст в Telegram группе
- Убедись, что он пришёл в VK с префиксом `[TG | ...]` и маркером `[BRIDGE:TG→VK|...]`

3) **Анти-цикл**
- Отправь в любой чат сообщение, содержащее `[BRIDGE:`
- Убедись, что оно не пересылается дальше

4) **Повтор webhook**
- Повтори отправку похожих сообщений и проверь, что нет “пинг‑понга” между чатами
